# Fitness Center Yönetim ve Randevu Sistemi

## Proje Hakkında
Bu proje, Web Programlama dersi kapsamında geliştirilen kapsamlı bir Spor Salonu (Fitness Center) Yönetim Sistemidir.
ASP.NET Core MVC mimarisi kullanılarak geliştirilmiştir

## Projenin Amacı
Web programlama dersinde öğrenilen teorik bilgilerin gerçek bir senaryo üzerinde uygulanması 
ve spor salonları için dijital bir çözüm üretilmesidir

## Özellikler
* **Spor Salonu Yönetimi:** Çalışma saatleri ve hizmet tanımları.
* **Antrenör Yönetimi:** Uzmanlık alanları ve müsaitlik durumu takibi.
* **Randevu Sistemi:** Üyeler için akıllı randevu oluşturma ve çakışma kontrolü.
* **Yapay Zeka Desteği:** Kişiye özel egzersiz önerileri
* **Raporlama:** REST API ile veri servis etme

## Kullanılan Teknolojiler
* ASP.NET Core MVC
* Entity Framework Core & LINQ
* SQL Server
* Bootstrap 5, HTML5, CSS3, JS
* OpenAI API (Yapay Zeka Entegrasyonu için)
